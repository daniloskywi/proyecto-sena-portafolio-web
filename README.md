# Portafolio Web - Danilo Rodriguez Diaz

Portafolio web personal desarrollado como proyecto integrador SENA.

## Tecnologías usadas
- HTML5, CSS3, JavaScript
- Node.js + Express
- MySQL

## Cómo ejecutarlo
1. Clonar el repositorio
2. Instalar dependencias: `npm install`
3. Crear archivo `.env` con las credenciales de la base de datos
4. Ejecutar: `node servidor.js`
